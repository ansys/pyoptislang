get_actor_registered_output_slots#

TcpOslServer.get_actor_registered_output_slots(uid: str, include_reference_values: bool = True) → List[dict]#

Get currently registered output slots for a certain (integration) actor.

Parameters:
uidstr

Actor uid.

include_reference_values: bool, optional

Whether reference values are to be included.

Returns:
List[dict]

Actor’s registered output slots.

Raises:
OslCommunicationError

Raised when an error occurs while communicating with server.

OslCommandError

Raised when the command or query fails.

TimeoutError

Raised when the timeout float value expires.
